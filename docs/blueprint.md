# **App Name**: Fé & Finanças

## Core Features:

- Multi-User Authentication: Secure user registration and login functionality, allowing multiple individuals to manage their tithe records independently.
- Monthly Income Entry: Users can easily record their monthly income entries, including amount and date, to be used for tithe calculations.
- Tithe Calculation Dashboard: Displays a clear overview of the current month's total income and automatically calculates the 10% tithe amount.
- Income History: Provides access to a historical record of all past monthly income entries and corresponding tithe calculations.
- Stewardship Reflection Tool: An AI-powered tool that generates personalized reflective prompts or relevant scriptural insights concerning financial stewardship.

## Style Guidelines:

- Light color scheme with a primary deep forest green (#3D6648), signifying growth, trustworthiness, and natural stewardship.
- Background color is a subtle, almost-white shade with a hint of green (#EFF4F1), providing a clean and calm canvas.
- Accent color is a vibrant spring green (#8AD16E), used for highlights and interactive elements to draw attention.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, precise, and tech-savvy feel suitable for titles and numerical displays.
- Body font: 'Inter' (sans-serif) for high readability and a neutral, objective tone across all text content, including descriptions and data entries.
- Use clean, minimalist line-art icons that are clear and universally understood, relating to finance, data entry, and user management.
- Clean and intuitive interface design, with clear separation between data input, summary views, and historical records to ensure ease of navigation and data comprehension.
- Subtle, smooth animations for user interactions such as navigation transitions and feedback upon form submissions, enhancing the user experience without distraction.
